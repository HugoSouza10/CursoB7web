REGEX FAZ BUSCAS E VERIFICAÇÕES

1- BUSCAS(PODEMOS BUSCAR POR PADRÕES)

2- VALIDAÇÕES(VERIFICA SE ATENDE AOS PADRÕES ESTABELECIDOS)

3- SUBSTITUIÇÕES(SUBSTITUI O PADRÃO)


[]: TUDO QUE É DIGITADO AI REPRESENTA UM CARACTER A SER BUSCADO

EX: [0] BUSCA O CARACTER 0

EX2: [02] BUCA O CARACTER 0 OU 2

EX3: [0-9] BUSCA TODOS OS NÚMEROS ENTRE 0 E 9

EX4: [0][0] BUSCA DOIS CARACTER 0


USANDO OS QUANTIFICADORES:

EX:[0]{2}: AS CHAVES INDICA QUANTAS VEZES VOCÊ QUER BUSCAR O 0

EX2: [0-9]{6}: BUSCA NÚMEROS ENTRE 0 E 9 QUE CONTÉM 6 CARACTERES
